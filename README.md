 CS-webTG v2.0

A full-stack web template generator project with:
- Static frontend pages (`frontend/`)
- React + Vite frontend (`frontend-react/`)
- Node.js/Express backend (`Backend/server/`)
- UI reference assets (`Website UI/`)

Tech Stack

- Frontend: HTML, CSS, JS, React, Vite
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)
- Integrations: Twilio Verify (OTP), OpenAI API
- Security: Helmet, CORS, Rate Limiting
